# 天津菜点餐系统（无图片版）

## 项目说明

这是一个完全无图片的天津菜点餐系统前端应用，采用纯文本展示方式，具有快速加载、简洁美观的特点。

## 文件结构

```
CD/
├── index.html      # 主页面文件
├── style.css       # 样式文件
├── script.js       # 逻辑脚本
└── README.md       # 说明文档
```

## 功能特性

### ✅ 核心功能

1. **菜品展示**
   - 156 道天津特色菜品
   - 6 个分类：全部、凉菜、热菜、汤品、主食、酒水饮料
   - 分类筛选功能
   - 搜索功能

2. **收藏功能**
   - 添加/取消收藏
   - 收藏列表展示
   - 收藏数量统计

3. **购物车功能**
   - 添加菜品到购物车
   - 数量调整（增加/减少）
   - 删除菜品
   - 清空购物车
   - 实时总价计算
   - 一键下单

4. **已点菜品**
   - 历史订单记录
   - 删除单条记录
   - 清空所有记录
   - 订单时间戳

5. **数据持久化**
   - 使用 localStorage 本地存储
   - 刷新页面数据不丢失
   - 支持长期保存

### 🎨 界面特点

- **无图片设计**：纯文本展示，加载速度极快
- **响应式布局**：适配移动端和桌面端
- **双列网格**：菜品列表采用双列布局
- **渐变配色**：橙红色渐变主题
- **动画效果**：悬停、点击、收藏等交互动画
- **底部导航**：4 个主要功能页面切换

## 使用方法

### 方式一：直接打开

1. 在文件管理器中找到 `index.html`
2. 双击打开（使用默认浏览器）
3. 即可使用

### 方式二：本地服务器

1. 使用 Python 启动简单 HTTP 服务器：
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   ```

2. 在浏览器中访问：
   ```
   http://localhost:8000
   ```

### 方式三：使用 VS Code Live Server

1. 安装 VS Code 的 Live Server 扩展
2. 右键点击 `index.html`
3. 选择 "Open with Live Server"

## 数据存储说明

应用使用浏览器的 localStorage 进行数据存储：

- `tianjin_favorites`：收藏的菜品 ID 列表
- `tianjin_cart`：购物车数据
- `tianjin_orders`：已点菜品历史记录

数据永久保存在本地，除非：
- 用户清除浏览器数据
- 使用无痕模式
- 手动删除 localStorage

## 常见问题

### Q: 为什么没有图片？
A: 这是特意设计的无图片版本，目的是提供最快的加载速度和最简洁的界面。

### Q: 数据会丢失吗？
A: 数据保存在浏览器本地存储中，只要不清除浏览器数据就不会丢失。

### Q: 可以在手机上使用吗？
A: 可以，界面采用响应式设计，完美适配手机、平板和电脑。

### Q: 如何清空所有数据？
A: 在浏览器开发者工具中执行：
```javascript
localStorage.clear();
location.reload();
```

## 开发说明

### 代码结构

**index.html**
- 页面结构和布局
- 4 个功能页面
- 底部导航栏

**style.css**
- 全局样式和重置
- 组件样式（卡片、按钮、列表等）
- 动画效果定义
- 响应式适配

**script.js**
- 菜品数据定义
- 页面渲染逻辑
- 事件处理
- 本地存储管理

### 主要函数

```javascript
// 初始化
init()

// 渲染函数
renderCategories()     // 渲染分类
renderDishes()         // 渲染菜品
renderFavorites()      // 渲染收藏
renderCart()           // 渲染购物车
renderOrders()         // 渲染已点菜品

// 交互函数
toggleFavorite(id)     // 切换收藏
addToCart(id)          // 加入购物车
updateQuantity(id, change)  // 调整数量
removeFromCart(id)     // 删除购物车项
deleteOrder(index)     // 删除订单

// 工具函数
updateBadges()         // 更新徽章
switchPage(page)       // 切换页面
searchDishes()         // 搜索菜品
```
