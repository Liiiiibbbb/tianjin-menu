# 📱 手机访问指南

## 问题解答

### Q1: 启动服务器.bat 是必须开启才能打开网站吗？

**答：不是必须的！**

- **自己使用**：直接双击 `index.html` 即可，无需启动服务器
- **手机访问**：需要启动服务器，让手机和电脑在同一 WiFi 下

---

## 📲 手机访问方法（推荐）

### 方法一：使用启动脚本（最简单）

#### 步骤 1：启动服务器
双击运行 `启动服务器.bat`

#### 步骤 2：查看访问地址
服务器启动后会显示访问地址，例如：
```
本地访问：http://localhost:8000
手机访问：http://192.168.31.102:8000
```

#### 步骤 3：手机访问
1. **确保手机和电脑连接同一 WiFi**
2. 打开手机浏览器（Safari/Chrome/UC 等）
3. 输入地址：`http://192.168.31.102:8000`
4. 即可访问！

---

### 方法二：手动启动服务器

#### 步骤 1：打开命令行
```
Win + R → 输入 cmd → 回车
```

#### 步骤 2：进入项目目录
```cmd
cd D:\卫宁文件\AI\LBY\CD
```

#### 步骤 3：启动服务器
```cmd
python -m http.server 8000
```

#### 步骤 4：手机访问
- 手机浏览器输入：`http://电脑IP:8000`

## 🔍 如何查看电脑 IP 地址

### 方法 1：使用命令查看

1. 按 `Win + R`
2. 输入 `cmd` 回车
3. 输入命令：
   ```cmd
   ipconfig
   ```
4. 找到 **IPv4 地址**，例如 `192.168.31.102`

### 方法 2：使用提供的工具

双击运行 `查看 IP 地址.bat`，会自动显示 IP 地址

## ⚠️ 常见问题

### 问题 1：手机无法访问

**解决方案：**

1. **检查 WiFi**
   - 确保手机和电脑连接**同一 WiFi**
   - 不能一个用 WiFi，一个用流量

2. **检查防火墙**
   - Windows 防火墙可能阻止访问
   - 临时关闭防火墙或添加例外规则

3. **检查 IP 地址**
   - 确保输入的 IP 地址正确
   - 重新运行 `查看 IP 地址.bat` 获取最新 IP

4. **检查端口**
   - 默认端口是 8000
   - 确保没有被其他程序占用

---

### 问题 2：启动服务器失败

**可能原因：**

1. **Python 未安装**
   ```
   解决方法：安装 Python
   下载地址：https://www.python.org/downloads/
   ```

2. **端口被占用**
   ```
   解决方法：更换端口
   修改启动服务器.bat 中的 8000 为其他端口（如 8080）
   ```

3. **权限问题**
   ```
   解决方法：右键 → 以管理员身份运行
   ```

---

### 问题 3：访问速度慢

**解决方案：**

1. **检查网络**
   - 确保 WiFi 信号良好
   - 减少其他设备占用带宽

2. **关闭其他程序**
   - 关闭占用网络的程序
   - 关闭不必要的浏览器标签

3. **使用 5G WiFi**
   - 连接 5G 频段 WiFi（如果支持）
   - 速度更快更稳定

---

## 🌐 远程访问（不在同一网络）

如果想让**不在同一 WiFi 的人也能访问**，需要：

### 方案一：使用内网穿透工具（推荐）

#### 使用 ngrok

1. **下载 ngrok**
   - 官网：https://ngrok.com/
   - 下载对应版本

2. **注册获取 token**
   - 注册免费账号
   - 获取认证 token

3. **配置 ngrok**
   ```cmd
   ngrok config add-authtoken 你的token
   ```

4. **启动 ngrok**
   ```cmd
   ngrok http 8000
   ```

5. **获取公网地址**
   - ngrok 会显示一个公网地址
   - 例如：`https://abc123.ngrok.io`
   - 任何人都可以通过这个地址访问

---
