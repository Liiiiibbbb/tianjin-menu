# GitHub 账号注册指南

## 📝 注册步骤

### 步骤 1：访问 GitHub 官网

**网址**：https://github.com

在浏览器中打开上述网址

---

### 步骤 2：点击注册按钮

在页面右上角找到并点击 **"Sign up"** 按钮

---

### 步骤 3：输入邮箱地址

在输入框中输入您的**电子邮箱地址**

**要求**：
- ✅ 有效的邮箱（QQ 邮箱、163 邮箱、Gmail 等都可以）
- ✅ 确保可以接收邮件（需要验证）

**示例**：
```
your-email@qq.com
your-email@163.com
your-email@gmail.com
```

点击 **"Continue"** 继续

---

### 步骤 4：创建密码

输入您的**登录密码**

**密码要求**：
- ✅ 至少 8 个字符
- ✅ 建议使用字母 + 数字组合
- ✅ 区分大小写

**示例**：
```
MyPassword123
Github2026!
```

点击 **"Continue"** 继续

---

### 步骤 5：设置用户名

输入您想要的 **GitHub 用户名**

**用户名要求**：
- ✅ 只能包含字母、数字和连字符（-）
- ✅ 不能以连字符开头
- ✅ 不能与现有用户重复
- ✅ 长度 1-39 个字符

**建议**：
- 使用有意义的用户名（如姓名拼音、昵称等）
- 避免使用复杂难记的用户名
- 这个用户名会出现在您的公网访问地址中

**示例**：
```
zhangsan
lisi2026
wangwu-dev
```

系统会自动检查用户名是否可用：
- ✅ 绿色勾号 = 可用
- ❌ 红色叉号 = 已被使用

点击 **"Continue"** 继续

---

### 步骤 6：邮箱验证

GitHub 会向您的邮箱发送**验证邮件**

**操作步骤**：
1. 打开您的邮箱
2. 找到来自 GitHub 的邮件（标题：Verify your email address）
3. 点击邮件中的验证链接
4. 或复制验证码输入到网页

**提示**：
- 如果收件箱没有，检查垃圾邮件箱
- 验证码有效期为 24 小时

---

### 步骤 7：安全验证

完成一个简单的**拼图验证**或**选择题验证**

**目的**：证明您是真人，不是机器人

点击 **"Verify"** 完成验证

---

### 步骤 8：个性化设置（可选）

GitHub 会问您一些问题来个性化您的体验：

**问题示例**：
- 您是学生还是专业人士？
- 您使用 GitHub 的频率？
- 您的工作经验？

**建议**：
- 这些都是可选的，可以跳过
- 点击 **"Skip"** 或直接关闭

---

### 步骤 9：注册完成！🎉

现在您已经成功注册 GitHub 账号！

**您将获得**：
- ✅ GitHub 账号（用户名 + 密码）
- ✅ 个人主页：`https://github.com/您的用户名`
- ✅ 免费代码托管空间
- ✅ GitHub Pages 免费网站托管

---

## 🔐 重要提示

### 1. 保存账号信息

**务必保存**：
- 用户名：`_______________`
- 密码：`_______________`
- 邮箱：`_______________`

**建议**：
- 记录在安全的地方
- 使用密码管理器
- 开启两步验证（推荐）

---

### 2. 开启两步验证（强烈推荐）

**为什么**：
- 提高账号安全性
- 防止账号被盗

**如何开启**：
1. 点击右上角头像 → Settings
2. 左侧菜单：Password and authentication
3. 找到 "Two-factor authentication"
4. 点击 "Enable two-factor authentication"
5. 按提示绑定手机或认证 APP

---

### 3. 记住个人主页地址

注册成功后，您的个人主页地址是：

```
https://github.com/您的用户名
```

**示例**：
```
https://github.com/zhangsan
```

这个地址很重要，稍后部署网站会用到！

## ❓ 常见问题

### Q1: 收不到验证邮件怎么办？

**解决方法**：
1. 检查垃圾邮件箱
2. 将 `@github.com` 加入邮箱白名单
3. 等待 5-10 分钟
4. 如果还是收不到，点击 "Resend email" 重新发送

---

### Q2: 用户名已被占用怎么办？

**解决方法**：
1. 尝试添加数字（如 zhangsan2026）
2. 尝试添加连字符（如 zhang-san）
3. 尝试不同的组合
4. 选择一个独特且容易记住的用户名

---

### Q3: 忘记密码怎么办？

**解决方法**：
1. 访问 https://github.com/password_reset
2. 输入注册邮箱
3. 点击 "Send reset email"
4. 查收邮件，按提示重置密码

---

### Q4: 可以修改用户名吗？

**答案**：可以，但不建议频繁修改

**修改方法**：
1. Settings → Account
2. 找到 "Change username"
3. 点击 "Change username"
4. 按提示操作

**注意**：
- 修改后，之前的访问地址会失效
- 需要更新所有相关链接

---

### Q5: 注册需要付费吗？

**答案**：完全免费！

GitHub 个人账号是**完全免费**的，包括：
- ✅ 无限公开仓库
- ✅ 无限私有仓库
- ✅ GitHub Pages 免费托管
- ✅ 基础功能全部免费
